# OLS-50 FE TC — เรียกดู คลังสื่อทั้งหมด

## Shared data preparation

1. เตรียมบัญชี Guest (ไม่ได้เข้าสู่ระบบ) และบัญชี Learner ที่พร้อมใช้งาน
2. ระบบมีสื่อสถานะ Published อย่างน้อย 15 รายการ ครอบคลุมหลายประเภทสื่อ, กลุ่มเป้าหมาย, หมวดหมู่, Tag
3. ระบบมีสื่อสถานะ Unpublished อย่างน้อย 1 รายการ และมีสื่อที่ถูกลบ (Deleted) อย่างน้อย 1 รายการ
4. เข้าถึงหน้าคลังสื่อทั้งหมดผ่านเส้นทาง: หน้าหลัก → คลังสื่อทั้งหมด

> **Precondition column:** หลังทำ Shared data preparation ครบแล้ว ให้ดำเนินการตาม Precondition ของแต่ละ TC ก่อนเริ่มขั้นตอนการทดสอบ
